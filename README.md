# Higher-Order Agency

Code accompanying the paper

> **When Does a Collection Become an Agent? Alignment and Identity Persistence in
> Multiscale Systems**
> Yuriria Cortés-Poza (IIMAS, UNAM).

A minimal multi-agent model in which a collection of locally goal-directed units
becomes a higher-order agent when their actions align toward the correction of a
collective error. The same framework captures three regimes: the **birth** of a
higher-order agent through a sharp dynamical transition, the **persistence** of its
functional identity through substrate turnover, and its **fragmentation** when parts
keep interacting but lose alignment (a minimal counterpart of cancer-like breakdown).

The central claim is that what distinguishes an agent from a merely integrated
collection is not coupling or integration but **cross-scale alignment** `Q`, and that
higher-order error correction `Ψ` (recovery beyond what isolated parts achieve) is its
operational signature.

## Installation

```bash
git clone https://github.com/<user>/higher-order-agency.git
cd higher-order-agency
pip install -r requirements.txt        # numpy, scipy, matplotlib
```

Python ≥ 3.9. No installation of the package itself is required; experiments add the
repository root to `sys.path`.

## Reproducing the paper

All results are deterministic (fixed random seeds). To regenerate every figure and
print the reported numbers:

```bash
python reproduce_all.py
```

Figures are written to `paper/figs/`. Each experiment is also runnable on its own.

## The model in brief

`N` units on a graph; unit `i` has a scalar state `x_i` and a fixed private goal
`g_i`. A collective descriptor `C(x) = mean(x)` has its own goal `G*`, chosen so the
collective goal cannot be met if every unit just pursues its own. Each unit carries a
**commitment** `s_i ∈ [0,1]` that evolves by a public-goods replicator rule: committing
is individually costly (`c`) but pays a benefit proportional to neighbourhood
commitment (`b·f_i`). This frequency dependence is the only essential nonlinearity and
produces a sharp transition at the critical benefit `b_c = 2c`.

Three quantities are measured (none is the swept control parameter):

- **`Q`** — cross-scale alignment: the (rescaled) fraction of units whose move reduces
  the collective error. `Q>0` parts serve the whole; `Q<0` they oppose it.
- **`Ψ`** — higher-order error correction: collective recovery from a perturbation in
  excess of the same units acting in isolation. `Ψ→1` marks the birth of an agent.
- **`B`** — boundary coherence; with the goal gap it forms the functional signature
  `σ = (|C−G*|, Q, R, B)` whose stability defines identity persistence (`D_self`).
